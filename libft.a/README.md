# libft

## Description
本プロジェクトはC言語における独自の汎用関数ライブラリ「Libft」の作成するものです。
- Part1：Libc関数の再実装
	- 文字判定・変換系
	- 文字列操作系
	- メモリ操作系
	- その他・動的メモリ割当系

- Part2：追加の汎用関数の実装
	- 動的メモリを伴う文字列操作
	- 数値から文字列への変換
	- 関数ポインタを使った反復処理
	- ファイルディスクリプタへの出力

- Part3：連結リスト操作
	- ノードの作成と追加
	- リストの探索
	- ノードの削除とメモリ開放
	- リストの反復処理

## Instructions
### 1.ライブラリのコンパイル
このライブラリを使用するには、まずコンパイルして'libft.a'というファイルを作成する必要があります。

bash
	make

### 2.自分のコードでの使い方
ライブラリの関数をC言語のファイルでヘッダーファイルをインクルードします。

c
	#include "libft.h"

自分のプログラムをコンパイルする際に作成した'libft.a'を一緒にコンパイルします。

bash
	cc main.a libft.a

### 3.ファイルの掃除
作成されたオブジェクトファイル（.o）を削除するには以下のコマンドを使用します。

bash
	make clean

オブジェクトファイルに加えて、作成された’libft.a’も完全に削除して初期状態に戻したい間合いは以下を実行してください。

bash
	make fclean
